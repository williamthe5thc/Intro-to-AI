Module 2 Practice: Zero and One-Shot Prompting Activity

Practice fundamental prompting techniques using real workplace scenarios. You'll experiment with different approaches and compare their effectiveness.

Activity Overview

Activity Type: Practical Prompting Exercise
Estimated Time: 30 minutes
Tools: Any AI assistant (ChatGPT, Claude, Copilot, etc.)

PART A: ZERO-SHOT PROMPTING (10 MINUTES)

Scenario: Professional Email Decline

You need to write a professional email declining a meeting invitation.

Task 1: Basic Prompt

Try this simple prompt and record the results:

"Write an email declining a meeting invitation."
Task 2: CLEAR Framework Prompt

Now try the same task using the CLEAR framework:

Context: I'm a project manager who needs to decline a weekly status meeting because of a scheduling conflict.

Length: Keep it under 100 words.

Examples: Use a polite but direct tone, similar to standard business correspondence.

Audience: My direct supervisor who scheduled the meeting.

Role: Act as a diplomatic professional who values both the meeting and their time constraints.

Write an email declining this meeting invitation.

Reflection

Compare the two outputs. What differences do you notice in tone, specificity, and professionalism?

PART B: ONE-SHOT PROMPTING (10 MINUTES)

Scenario: Executive Report Summary

You need to summarize a lengthy report for executives.

Task 3: Prompt with Example

Provide this prompt with an example:

I need to create executive summaries from detailed reports. Here's the format I want:

Example Summary:
**Key Finding:** Revenue increased 15% due to new customer acquisition
**Impact:** $2.3M additional quarterly revenue
**Recommendation:** Expand marketing budget by 20% to accelerate growth
**Timeline:** Implement within Q1 for Q2 results

Now create a similar summary for this report: [paste your own work report or use provided sample]

Analysis

How did providing an example change the quality and format of the output?

PART C: ITERATION PRACTICE (10 MINUTES)

Scenario: Presentation Outline Creation

Creating a presentation outline through iterative refinement.

Task 4: Iterative Refinement

Start with a basic prompt, then refine it through 2-3 iterations:

Initial Prompt:

"Create a presentation outline about artificial intelligence."

Your Iterations:

  • Iteration 1: Add specificity about audience and context
  • Iteration 2: Include timing and key message requirements
  • Iteration 3: Specify format and style preferences

Documentation

Record each prompt and its output to see how iteration improves results.

Submission Requirements

Submit a document containing:

  1. All prompts you used
  2. The AI outputs for each prompt
  3. Your reflection on which approaches worked best
  4. At least 2 specific improvements you noticed when using the CLEAR framework

Assessment Criteria

Prompt Quality (40%)

Use of CLEAR framework principles

Analysis (30%)

Thoughtful comparison of different approaches

Professional Application (20%)

Relevance to workplace scenarios

Reflection (10%)

Insights about effective prompting

Extension Challenge

Choose a task from your actual work and create 3 different prompts for it using different techniques covered in this module. Compare the results and identify which approach works best for your specific professional context.
